## ようこそ！

これは **pitcms** と **Next.js** で構築されたブログの最初の記事です。

### pitcms とは？

pitcms は Git ベースのヘッドレス CMS です。すべてのコンテンツが GitHub リポジトリに保存されるため、ベンダーロックインの心配がありません。

### 主な特徴

- **Git ベース**: コンテンツはすべて GitHub に保存
- **Web UI**: 非エンジニアでも快適に編集可能
- **プレビュー機能**: 公開前に変更を確認
- **AI 編集**: テキストの要約・翻訳・校正をワンクリックで

### 使い方

1. `pitcms.jsonc` でコレクションを定義
2. pitcms ダッシュボードでコンテンツを編集
3. 自動で GitHub にコミット＆プッシュ
4. Next.js が自動でビルド＆デプロイ

コンテンツ管理を始めましょう！
